# Propaganda Detection in Social Media

## Overview
This project develops a machine learning model to detect propaganda in tweets using linguistic and structural features.

---

## Dataset
- 30,000 labelled tweets
- Binary classification: propaganda vs non-propaganda

---

## Approach

### Feature Engineering
- TF-IDF vectorisation
- Sentiment scores
- Profanity detection
- Rhetorical patterns
- Text structure

### Model
- Logistic Regression classifier

---

## Results
- Accuracy: 87%
- F1-score (non-propaganda): 0.99

---

## Additional Study
- Survey with 50+ participants
- Explored user attitudes towards misinformation tools
- 59.6% expressed interest in detection systems

---

## Tech Stack
- Python
- Scikit-learn
- NLP preprocessing

---

## Key Insights
- Feature-based models remain highly effective for structured tasks
- Interpretability is valuable for real-world deployment
